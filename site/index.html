Provider-neutral agent auth

Fresh user tokens at the exact moment an agent acts.

nominee is the identity and token delegation layer for AI agents: fetch fresh third-party access tokens, pause for human approval when needed, and keep an audit trail across frameworks.

npm i nominee Read the docs
tool-call.ts 
import { Nominee, tokens } from 'nominee'

const nominee = new Nominee({
  strategy: tokens(({ user, connection }) =>
    db.getFreshToken(user, connection)
  ),
  agent: 'triage-bot',
})

const token = await nominee.token({
  user: 'alice',
  connection: 'github',
})

Token flow

Never hand a captured token to a durable agent.

Agents ask nominee for a token at the moment of a tool call. nominee returns the cached token only when it is still fresh; otherwise it asks your strategy for a new one.
